## install Jekyll

```
sudo gem install jekyll
sudo gem install jekyll-feed jekyll-sitemap jekyll-paginate
```

## Run Jekyll Locally

cd to the working folder, then
```
jekyll serve
```

## push

```
git add .
git commit -m "commit msg here"
git push
```
